Introduction to Python

Learn the basics of Python programming and why it's one of the most popular languages.

Getting Started with Python

Python is a powerful, easy-to-learn programming language that is widely used in web development, data science, AI, and automation.

Why Learn Python?

  • Simple and easy to read syntax.
  • Great for beginners and experts alike.
  • Used in web development, machine learning, data science, and more.
  • Large community and plenty of resources.

Installing Python

You can download Python from the official website: python.org.

Your First Python Program

Let's write a simple Python program that prints "Hello, World!" to the screen. 

print("Hello, World!")

Python Variables

In Python, you can create variables without specifying their type. 

name = "John"
age = 25
print(name, age)

Basic Data Types in Python

  • Integers: Whole numbers (e.g., 10, 50, -3)
  • Floats: Decimal numbers (e.g., 3.14, -0.5)
  • Strings: Text data (e.g., "Hello")
  • Booleans: True or False values

Conditional Statements

Python allows you to make decisions using if, elif, and else. 

age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

Loops in Python

Use loops to repeat code multiple times. 

for i in range(5):
    print("Iteration:", i)

This is just a basic introduction to Python. Stay tuned for more advanced Python tutorials!
