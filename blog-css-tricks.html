Enhance your website styling with these advanced CSS techniques.

Top 5 CSS Tricks to Improve Your Website

1. CSS Variables

CSS variables allow you to define reusable values across your stylesheets. 

:root {
    --primary-color: #3498db;
}
body {
    background-color: var(--primary-color);
}

2. Flexbox for Easy Layouts

Flexbox helps create responsive layouts with minimal CSS. 

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

3. CSS Grid for Advanced Layouts

CSS Grid is a powerful way to design complex layouts. 

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

4. Hover Effects for Better UX

Adding hover effects makes your site more interactive. 

button:hover {
    background-color: #2ecc71;
    color: white;
}

5. Dark Mode with CSS

Implement dark mode easily using CSS. 

@media (prefers-color-scheme: dark) {
    body {
        background-color: #121212;
        color: white;
    }
}
