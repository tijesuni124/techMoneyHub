Deep Dive into JavaScript

JavaScript is a versatile language used for web development, backend programming, and even mobile applications.

Why Master JavaScript?

  • Build interactive websites and web applications.
  • Work with modern JavaScript frameworks like React, Vue, and Angular.
  • Create backend applications with Node.js.
  • JavaScript is in high demand for web development jobs.

JavaScript Variables

JavaScript has three main ways to declare variables: var, let, and const. 

var name = "John";  // Old way (avoid using)
let age = 25;       // Block-scoped variable
const PI = 3.14;    // Constant value (cannot be changed)

Functions in JavaScript

Functions allow you to reuse blocks of code efficiently. 

// Regular function
function greet(name) {
    return "Hello, " + name;
}

// Arrow function
const greetArrow = (name) => "Hello, " + name;

console.log(greet("Alice"));

Asynchronous JavaScript (Promises & Async/Await)

JavaScript is single-threaded, but you can handle asynchronous operations using Promises and async/await. 

// Using Promises
fetch('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// Using Async/Await
async function getData() {
    try {
        let response = await fetch('https://jsonplaceholder.typicode.com/posts/1');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}
getData();

JavaScript ES6+ Features

ECMAScript 6 (ES6) introduced new features that make JavaScript more powerful and readable.

  • Arrow Functions → Shorter syntax for functions.
  • Template Literals → Use backticks (`) for string interpolation.
  • Destructuring → Easily extract values from objects and arrays.
  • Spread & Rest Operators → Useful for merging and handling arrays.
  • Modules → Use import and export to organize code.

JavaScript DOM Manipulation

You can use JavaScript to manipulate the Document Object Model (DOM) and make web pages dynamic. 

// Changing text content
document.getElementById("demo").textContent = "Hello, JavaScript!";

Event Listeners

Use event listeners to handle user interactions. 

document.getElementById("btn").addEventListener("click", function() {
    alert("Button Clicked!");
});
